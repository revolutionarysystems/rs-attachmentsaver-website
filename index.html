Features

group

Specify your senders

Simply set up your list of allowed senders. Anyone else trying to send will be silently ignored.
For added peace of mind, AttachmentSaver also incorporates powerful spam and virus scanning, so if your allowed senders have their email compromised, bad files will not be allowed through.

insert_drive_file

Define your file types

Just specify the file extensions you are happy to accept. Your senders can be politely notified if they try to save something else.
AttachmentSaver supports multiple attachments - typically as many and as large as your email service will permit.

folder

Point to your repository

Easily specify the repository and location within it where the attachments should be placed.
AttachmentSaver even lets you set up multiple email addresses for receiving files, each one of which can be pointed to a different location.
